# Roadmap
## 1.0.0
* 多k8s集群管理
* 实现k8s上管理部署大数据服务
* 节点管理
* 服务角色重启、启动、停止
* 服务重启、启动、停止、配置、删除
* 服务告警提示、服务dashboard
* 服务配置
* 指令记录详情
* 服务角色实时日志
* 告警规则，活跃告警，历史告警
* 支持hdfs、yarn、zookeeper、doris、hive、spark、flink、dolphinscheduler、hbase、kafka、prometheus、grafana、alertmanager等服务在Kubernetes上运行

* 更聪明的安装服务分配节点机制
## 1.2.0
